The six projects, in one line each

Intelligent Intake

Natural language in, structured fields out — no long forms.

Smart OCR Pipeline

Document type drives the schema, not the other way round.

Database-Centric AI

DB → schema → validators → rules → AI. The model queries structure, it doesn't author it.

AI + DB Bridge

One data layer, several models — no single point of lock-in.

Time & Work Intelligence

Classifies activity — compensable, training, admin, technical, legal — from context.
